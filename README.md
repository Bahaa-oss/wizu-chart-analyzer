# WIZU - AI-Powered Smart Money Chart Analyzer

A premium trading chart analyzer that uses GPT-4 Vision to provide institutional-grade technical analysis based on Smart Money Concepts.

## Features
- Upload chart screenshots (PNG, JPG)
- AI-powered analysis using GPT-4 Vision
- Smart Money Concepts: Order Blocks, Fair Value Gaps, Liquidity Levels
- Bullish/Bearish/Neutral bias with confidence score
- Key support and resistance levels
- Pattern detection
- Actionable trade recommendations
- Beautiful dark-themed premium UI with animations

## Quick Start
1. Clone the repo
2. Copy .env.example to .env and add your OpenAI API key
3. npm install
4. npm run dev

## Deployment
- Build: npm run build
- Deploy the dist/ folder to Vercel, Netlify, or any static host
- Set VITE_OPENAI_API_KEY as environment variable

## Tech Stack
- React 18 + TypeScript + Vite
- Framer Motion (animations)
- Lucide React (icons)
- OpenAI GPT-4 Vision API
